PDF LaTeX Editor

Link to Github repository

What is this for?

Currently there are no programs where you can add LaTeX formulas to a pdf while keeping the pdf as a pdf file. Other programs right now first convert the pdf to an image before you can add LaTeX formulas. This program provides a simple GUI where you can load multipage pdfs and add LaTeX formulas to them.

Wait, what's LaTeX?

LateX is a document preparation system that formats text for the author. It is especially great math and science, for example formatting an equation. For more information, see the LaTeX website .

How to download?

  1. Go to Github repository linked above and and download the folder

  2. Make sure you have Python 3.9 installed. If not, you can get it at the Python website .

  3. Make sure you have all needed modules (you can check by opening up PDFLaTeXEditor.py in IDLE and check under Dependencies. For each ModuleName (of form import ModuleName), you can install it by doing pip install ModuleName. If it exists, then it won't do anything.

  4. Set up your export and import paths (importDir will be where you want the program to open your PDFs and output the .duo files to, exportDir is where you want the program to output the edited PDFs at). Also set up the inkscapePath, which is the path where inkscape.exe is; drawPath, which is the path where LibreOfficeDrawPortable.exe is; and libPath, which is where the lib folder in PDFLaTeXEditor is.

How do I use it?

  1. Open PDFLaTeXEditor.py with Python. You should see a simple GUI with a blank PDF with a "Load" and "Open" button on the left, and "Save" and "Export" buttons under them.

  2. You can press "Load" to open up a file dialog to select a file. You can choose a PDF file to convert into a duo file, or open a converted .duo file. After selecting, you press "Open" to open it.

  3. On the left, there is a listbox where you can select the page you want to go to.

  4. Once you select a page, you can open it in LibreOffice Draw by pressing Ctrl+o.

  5. Once in LO Draw, you can add LaTeX boxes by clicking the blue "Ï€" sign in the corner. For more information, refer to here .

  6. In LO Draw, you must save your changes by clicking Ctrl+s. Once you're done with a page, you can go back to the main PDFLaTeXEditor tool by simply exiting the LO page.

  7. Once you're done with a document, click the "Save" button and it will go back to its "blank page" state. You can also close the window and the tool will automatically save the file for you.

  8. You can export your new edited file into a PDF by simply pressing the "Export" button and choosing what the file should be saved as.
